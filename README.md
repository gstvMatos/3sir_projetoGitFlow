## Descrição

Projeto desenvolvido para a Global Solution FIAP com foco na análise de queimadas no Brasil utilizando Ciência de Dados e Machine Learning.

## Objetivo

Identificar padrões espaciais e temporais relacionados às queimadas no Brasil a partir de dados satelitais disponibilizados pelo INPE.

## Tecnologias

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## Base de Dados

Dados Abertos INPE:
https://data.inpe.br/queimadas/dados-abertos/

## Principais Resultados

- Crescimento de 46,55% dos focos entre 2023 e 2024;
- Redução significativa em 2025;
- Amazônia e Cerrado como biomas mais afetados;
- Identificação de sazonalidade entre agosto e outubro;
- Clusterização geográfica das regiões críticas;
- Modelo Random Forest com elevada precisão na classificação dos biomas.
